# Images

No image uploads are required. This site uses:

- **Author photo:** Wikipedia Commons (Fredrik Backman at Bokmässan 2013).
- **Book covers:** [Open Library Covers API](https://openlibrary.org/dev/docs/api/covers); book thumbnails link to each book’s Open Library page.
- **Background:** A CSS-only gradient that evokes a Scandinavian / *My Friends*–style painted look.

To run the site, open `index.html` in a browser. No local image files are needed.
